# 全球用工智能问答项目小结

🌍 一个基于知识库 + AI 的全球用工政策智能问答系统

## 项目简介

本项目爬取了 Horizons 出海用工指南的 43 个国家/地区的用工政策文章，构建向量知识库，并提供智能问答服务。

## 技术栈

- **爬虫**: Python + pyppeteer
- **知识库**: ChromaDB + OpenAI Embedding
- **AI 模型**: Deepseek / OpenAI / Claude
- **Web 服务**: Flask
- **前端**: 原生 HTML + CSS + JavaScript

## 核心功能

✅ 43 个国家用工政策知识库  
✅ 智能语义检索  
✅ AI 生成结构化答案（四部分）  
✅ Deepseek 联网搜索兜底  
✅ 精美绿色主题 UI
